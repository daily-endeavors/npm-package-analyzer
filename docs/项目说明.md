#   项目说明

##  项目背景

本仓库为[第六届青训营前端大项目1 - NPM Package 分析工具](https://bytedance.feishu.cn/docx/MQHgdP03Ko8oUpxRnTnchUH7nEg)的具体实现

##  项目目标

项目的预期查看人为面试官, 期望团队成员在项目中的表现可以达到令面试官满意的标准. 每人在项目中的经历可以整理出至少3个STAR方案.

-   前端界面可以根据模拟json配置渲染具体界面

-   npm包发布完毕
